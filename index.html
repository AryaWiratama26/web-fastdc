Overview

FastDC is a lightweight and powerful Python library to build modern Discord bots with ease. It supports command handling, AI interaction, training, and more.

Features

  • Auto Command Handler
  • AI Chat Support (Groq API)
  • Trainable ChatBot with data_train.txt
  • Member Join & Leave Notifications

Installation

pip install fastdc

Usage

from fastdc import FastBot

bot = FastBot(token="YOUR_DISCORD_TOKEN")

bot.auto_reply(trigger="hi", response="Hello!")
bot.ai_chat(api_key_usr="YOUR_GROQ_API_KEY")
bot.train_bot()  # Make sure 'data_train.txt' exists
bot.welcome_member() # Sends "Hello {User}, Welcome to Server!"
bot.leave_member() # Sends "{User} has left the server üñêÔ∏è"
bot.run()
