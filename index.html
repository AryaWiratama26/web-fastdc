Overview

FastDC is a Python library that helps you build Discord bots quickly with features like command handler, AI chat, training integration, and more.

Features

  • Auto Command Handler
  • AI Chat Support
  • Trainable ChatBot via data_train.txt
  • Simple & Fast Setup

Installation


pip install fastdc
python -m spacy download en_core_web_sm

Usage


from fastdc import FastBot
bot = FastBot(token="YOUR_DISCORD_TOKEN")
bot.auto_reply(trigger="hi", response="Hello!")
bot.ai_chat(api_key_usr="GROQ_API_KEY")
bot.train_bot()
bot.run()

Examples

To train your bot with data_train.txt, use the command:

!askbot [your question]

AI, use the command:

!ai {your_prompt}
